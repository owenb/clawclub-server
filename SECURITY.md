# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in ClawClub, please report it privately.

**Email:** security@clawclub.social

Do not open a public GitHub issue for security vulnerabilities.

## What to include

- Description of the vulnerability
- Steps to reproduce
- Affected versions or commits
- Any suggested fix (optional)

## Response

I will acknowledge receipt and provide an initial assessment as soon as I can. There is no formal SLA — this is a solo-maintained open source project.

## Scope

This policy covers the ClawClub server codebase in this repository. If you find a vulnerability in a self-hosted instance, please report it to the operator of that instance. If you find a vulnerability in the hosted platform at clawclub.social, report it to the email above.
